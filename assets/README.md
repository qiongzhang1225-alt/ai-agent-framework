# assets/ — 自定义视觉资源

把图片放到这里，应用启动时自动加载，不需要改代码。

## 支持的文件

| 文件名                                           | 作用            | 推荐规格                    |
| --------------------------------------------- | ------------- | ----------------------- |
| `icon.png`                                    | 浏览器标签 / 任务栏图标 | 方形 64×64 或以上，PNG 透明背景最佳 |
| `background.png` / `.jpg` / `.jpeg` / `.webp` | 整个应用的背景图      | 1920×1080 及以上；深色更耐看     |
| `sidebar_bg.png` / `.jpg` / `.jpeg` / `.webp` | 侧边栏独立背景（可选）   | 任意尺寸，会自动 cover          |
|                                               |               |                         |

## 行为约定

- **文件不存在时跳过**，不会报错。
- 主背景启用时，聊天气泡会自动加半透明白色遮罩（`rgba(255,255,255,0.85)` + 轻微模糊），保证文字清晰。
- 主背景使用 `background-attachment: fixed`，滚动时背景不动。
- 检测顺序：`png` → `jpg` → `jpeg` → `webp`，取第一个找到的。

## 调整外观

如果默认遮罩不符合你的审美（比如背景图较暗、想要黑色文字背景），改 [app.py](../app.py) 里 `_inject_custom_style()` 函数中 `rgba(255, 255, 255, 0.85)` 这段 CSS 即可。
