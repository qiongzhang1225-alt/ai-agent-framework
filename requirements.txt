# 为什么锁死而不是用 >=：
#   之前全用 >=，新装的人会拿到上游"当下最新版"。曾因此踩坑——
#   新机器装到 transformers 5.12 / huggingface_hub 1.19，比稳定环境（5.9 / 1.16.4）新，

#   钉死到"已知能跑"的整套版本，是让别人下载即用最稳的办法。
#
# 如果 pip 在你的 Python / 系统上找不到某一行的 wheel（比如你用的不是 3.11）：
#   只把报错的那一行松成 >=，逐行试。但记忆栈（下面标了 ★ 的几行）尽量别动——
#   动了中文长期记忆可能不报错地失效，最难排查。
#
# 建议：用 Python 3.11.x 装最省事（这套版本就是在 3.11 上验证的）。
